# Smoothly  
这是一个第一人称策略型沙盒游戏，使用qt开发，图形基于irrlicht，物理基于bullet，网络基于raknet，数据库基于leveldb。

## 编译  
在编译前，请先安装qt,irrlicht,bullet,raknet,leveldb，然后进入src目录，执行`qmake && make`即可  
## 使用  
服务器端直接运行即可。管理员客户端和游戏客户端均具有GUI界面。
